# ServiceNow Developer Course

Welcome to the **ServiceNow Developer Full Course** — a hands-on path from JavaScript fundamentals to production-grade platform architecture, taught by [TechTalk with Bill](https://www.youtube.com/@techtalkwithbill).

Every lesson pairs a video with an overview, key concepts, and hands-on exercises you complete in your own free ServiceNow Personal Developer Instance (PDI).

{% embed url="https://www.youtube.com/watch?v=fXbSNOMOj7M" %}

## What you'll learn

- JavaScript and the ServiceNow execution model
- Client-side scripting: Client Scripts, UI Policies, Catalog scripting
- Server-side scripting: GlideSystem, Business Rules, GlideRecord
- Reusable code: Script Includes and GlideAjax
- Architecture: OOP, dependency injection, event-driven and service-bus patterns
- Integrations, secure coding, and global error handling
- A capstone project and your certification/career roadmap
