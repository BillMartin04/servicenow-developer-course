# How to use this course

## Prerequisites

- A free [ServiceNow Personal Developer Instance](https://developer.servicenow.com) (covered in [Module 0](module-0-getting-started/04-set-up-your-pdi.md)).
- Basic comfort with a web browser and a willingness to write code.
- No prior ServiceNow experience required — [Module 1](module-1-foundations/README.md) starts from JavaScript basics.

## Recommended pace

| Week | Modules |
| --- | --- |
| 1 | Module 0 + Module 1 |
| 2 | Module 2 (client-side) |
| 3 | Module 3 (server-side) |
| 4 | Module 4 (Script Includes & GlideAjax) |
| 5 | Module 5 (architecture) |
| 6 | Module 6 (integrations & reliability) |
| 7 | Module 7 (capstone) |

## Lesson structure

Every lesson follows the same shape:

1. **Watch** — the video for the topic.
2. **Overview** — what the lesson is about.
3. **Key concepts** — the ideas to internalise.
4. **Hands-on** — a checklist you complete in your PDI.
5. **Resources** — official docs and links.
