## Overview

Every hands-on exercise in this course needs a free ServiceNow Personal Developer Instance. This lesson walks through requesting, waking, and configuring your PDI.

## Key concepts

- Requesting a PDI from developer.servicenow.com
- Understanding instance lifecycle (hibernation, wake, reset)
- Enabling the JavaScript debugger and script logs
- Loading demo data and creating a scratch scoped app

## Hands-on
