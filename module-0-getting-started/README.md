# Module 0 · Getting Started

**Estimated time:** ~45 min  ·  **Lessons:** 3

Understand what a ServiceNow developer does, how this course works, and get a fully configured Personal Developer Instance ready for every lab.

## Learning objectives

By the end of this module, you will be able to:

- Describe the ServiceNow developer role and the skills this course builds
- Navigate the course structure and know how to study effectively
- Provision, wake, and configure a free Personal Developer Instance (PDI)
- Create a scoped lab application and verify your environment with a script

## Prerequisites

None. This is the starting point — no prior ServiceNow experience required.

## Knowledge check

Before moving on, make sure you can answer these:

- What is a PDI and why does this course use one?
- What happens to a PDI after ~10 days of inactivity, and how do you recover it?
- Why create a dedicated scoped application for your lab work?
