# Syllabus

A complete, self-paced path from JavaScript fundamentals to production-grade ServiceNow architecture. This page is your map: what you'll achieve, who it's for, what you need, and how the eight modules fit together.

## Course goal

> Become a **job-ready ServiceNow developer** who can build clean, secure, well-architected applications — not just scripts that happen to work.

## Learning outcomes

By the end of this course you will be able to:

- Write correct, platform-safe JavaScript and know exactly where each script runs.
- Build client-side experiences with Client Scripts and UI Policies.
- Manipulate data on the server with GlideSystem, Business Rules, and GlideRecord.
- Package reusable logic in Script Includes and call it from the client with GlideAjax.
- Apply professional architecture: OOP, dependency injection, event-driven design, and a service-bus integration layer.
- Integrate securely with external systems over REST and handle errors globally.
- Ship a production-quality capstone application and plan your certification and career path.

## Who this course is for

- Aspiring ServiceNow developers starting from little or no platform experience.
- Admins moving into development who want real scripting and architecture depth.
- Developers from other stacks mapping their skills onto ServiceNow.

## Prerequisites

- A free [ServiceNow Personal Developer Instance](https://developer.servicenow.com) — you'll set this up in Module 0.
- Willingness to write code and complete the hands-on exercises.
- No prior ServiceNow experience required; JavaScript is taught from the ground up in Module 1.

## How the course is structured

Eight modules, each building on the last. Every module has **learning objectives**, **prerequisites**, a **lesson table with time estimates**, a **knowledge check**, and a **recap**. Every lesson has objectives, a video or written guide, key concepts, hands-on exercises for your PDI, and resources.
